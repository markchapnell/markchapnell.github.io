HTML Cheat Sheet

Basic Structure

Tag Name Description
<!DOCTYPE html> HTML Declaration Tells the browser to expect an HTML file.
<html> HTML Tag Wraps the entire content of the HTML file.
<head> Header Section Contains information such as page title, link to CSS file etc.
<body> Page Body Wraps the main body of the page.
<foot> Page Footer Wraps the footer section of the page.
<div> Page Divide Creates a container for content that can be styled uniquely.
<section> Section Defines a section of the page that can be styled uniquely.
<article> Article Allows an article to be defined on a Blog page.
<h1> to <h6> Headers Defines multiple levels of headers that can each be styled.
<p> Paragraph Defines a paragraph of text.
<br> Line Break Moves text to the next line.
<img> Image Inserts an image.
<video> Video Inserts a video.

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Header The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table cell container.
<tfoot> Table Footer The set of rows defining the footer in a table.

Lists

Tag Name Description
<ul> Unordered List Creates a non-numerical list that by default is bulleted.
<ol> Ordered List Creates a numerically ordered list.
<li> Line Item Wraps each item in both an ordered and unordered list.
